# RepoResume.ai

An AI-powered resume and cover letter generator that analyzes your GitHub repositories to create tailored application materials. Connect your GitHub account, provide a job description, and generate professional resumes and cover letters in seconds with AI-selected relevant projects.

## Tech Stack

**Frontend:** React 18, TypeScript, Vite, React Router, Tailwind CSS, Lucide Icons  
**Backend:** Vercel Serverless Functions, Supabase (PostgreSQL, Auth, Storage)  
**AI:** OpenAI GPT-4o  
**Integrations:** GitHub API  
**PDF Generation:** jsPDF, html2canvas  
**Deployment:** Vercel

## Key Features

- **GitHub Integration:** Analyze up to 100 repositories and automatically select the most relevant projects
- **AI-Powered Generation:** Generate tailored resumes and cover letters using GPT-4o
- **Smart Project Selection:** AI matches your projects to job descriptions for maximum relevance
- **PDF Export:** Download professional PDFs of your resume and cover letter
- **Profile Management:** Manage education, work experience, and contact information
- **Rate Limiting:** 10 free generations per month with automatic reset
